# CloudSec Playbook

## Typical scope
- Cloud posture management and guardrails.
- IAM and networking hardening in cloud providers.
- Misconfiguration triage and drift control.

## Common requests
- Investigate exposed buckets, permissive security groups, risky roles.
- Recommend remediation for CSPM findings.
- Route to ticket workflow for tracked fixes.

## Inputs to classify
- Alerts and findings from cloud misconfiguration.
- Terms: aws, azure, gcp, bucket, security group, cloudtrail, iam role, cspm.

## Expected outputs
- Root cause and blast radius summary.
- Remediation plan and ticket handoff.
