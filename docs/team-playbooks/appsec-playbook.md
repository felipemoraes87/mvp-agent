# AppSec Playbook

## Typical scope
- Secure SDLC, code security, threat modeling.
- SAST, SCA, API security and dependency risk.
- Security review support for releases.

## Common requests
- Analyze vulnerabilities in code and dependencies.
- Recommend fixes and compensating controls.
- Route actionable findings to tracked tickets.

## Inputs to classify
- Scanner findings, code review issues, insecure API patterns.
- Terms: sast, sca, xss, sqli, dependency, cve, owasp, api auth.

## Expected outputs
- Severity and exploitability assessment.
- Prioritized remediation path.
