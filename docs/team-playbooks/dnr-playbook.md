# Detection and Response Playbook

## Typical scope
- SOC triage, incident response and containment.
- Alert enrichment and escalation management.
- Coordination with forensics and post-incident actions.

## Common requests
- Triage alerts and suspicious events.
- Recommend containment and eradication actions.
- Open and track incident tickets.

## Inputs to classify
- SIEM alerts, IOC hits, suspicious process/network activities.
- Terms: alert, incident, ioc, siem, soc, malware, containment, triage.

## Expected outputs
- Incident status and severity.
- Response actions and ticket instructions.
