# IAM IGA Playbook

## Typical scope
- Identity lifecycle (joiner/mover/leaver).
- Access reviews, segregation of duties, entitlement governance.
- Privileged access and recertification support.

## Common requests
- Revoke access, grant role-based access, reset and lock accounts.
- Validate SoD conflicts and toxic combinations.
- Open governance tickets for approvals and remediation.

## Inputs to classify
- Access requests, orphan accounts, privilege misuse, recertification tasks.
- Terms: access, entitlement, role, SoD, recertification, revoke, provisioning.

## Expected outputs
- Access decision recommendation.
- Ticket proposal with least privilege rationale.
