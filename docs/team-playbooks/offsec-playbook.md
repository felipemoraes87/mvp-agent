# OffSec Playbook

## Typical scope
- Adversarial testing and offensive validation.
- Pentest planning, exploit analysis, attack simulation.
- Validation of detection and hardening controls.

## Common requests
- Assess exposed assets and exploitable paths.
- Simulate attack chains and report critical gaps.
- Escalate severe findings for immediate remediation.

## Inputs to classify
- Pentest tasks, exploit PoC, external exposure findings.
- Terms: pentest, exploit, red team, attack path, lateral movement, recon.

## Expected outputs
- Attack narrative and risk score.
- Actionable remediation recommendations.
